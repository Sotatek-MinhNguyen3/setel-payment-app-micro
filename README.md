## Description

An payment microservice app to verify request from order app randomly

## Installation
yarn

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
